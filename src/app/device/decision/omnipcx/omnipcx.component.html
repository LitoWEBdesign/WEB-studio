Alcatel-Lucent OmniPCX Office

от 5 до 236 абонентов

Интегрированная система связи для малых и средних предприятий (SMB), Alcatel-Lucent OmniPCX Office предлагает персонального коммуникационного менеджера (PIMphony), передовые решения для контакт центра (Alcatel-Lucent OmniTouch Call Center Office) безопасный и высокоскоростной доступ в интернет, комплексные решения по TDM и IP-телефонии а также IP-инфраструктуре (LAN, WLAN). Решение Alcatel-Lucent состоит из предварительно настроенного сервера, который предоставляет каждому сотруднику услуги голосовой связи, доступ в Интернет, электронную почту и передачу данных.

Компоненты разработанные для OmniPCX Office
Alcatel-Lucent Extended Communication Server

Простое и модульное решение, которое отвечает всем требованиям малых и средних компаний для общения, взаимодействия и мобильности их сотрудников. Это решение позволяет сотрудникам получить максимум эффективности от поставленных перед ними бизнес задач, при помощи набора коммуникационных инструментов.

Alcatel-Lucent OmniTouch My Teamwork Office Edition (командная работа и конференции)

Программное обеспечение основанное на мульти-медиа, идеальное решение для бизнес-коммуникаций, которое работает на любом устройстве. Используйте OmniTouch My Teamwork Office Edition с любого телефона, в любом месте, безопасная конференция с любого браузера с кем угодно, внутри компании или за её пределами, без специализированного программного обеспечения и VPN сети. Индикация присутствия на рабочем месте, простой и понятный интерфейс, поддержка полного набора функций, конференции, чат, приложения для работы на удалённом рабочем столе, управление документами и много другое.

Alcatel-Lucent PIMphony

PIMphony – персональный коммуникационный менеджер, программный продукт который связывает настольный телефон с компьютером, и становится вашим надёжным помощником для выполнения повседневных задач. Он предлагает комплекс сервисов в который входит: история звонков , голосовая почта и различные виды мониторинга. Помощник постоянно доступен на вашем компьютере, в не зависимости от запушенных приложений. База контактов вашей компании полностью интегрируется с PIMphony для оптимизации работы сотрудников. PIMphony улучшает взаимодействие с клиентами, тем самым увеличивая продуктивность предприятия и персонала, оправдывая инвестиционные вложения в телекоммуникации.

Alcatel-Lucent PIMphony Attendan

приложения для централизованного управления звонками. Эргономичный интерфейс позволяющий управлять звонками и приветствием клиентов с быстрой переадресацией, моно или мульти администрирование.

Alcatel-Lucent OmniTouch Call Center Office (до 10 агентов)

Создан для малых предприятий, по принципу «всё в одном», предназначен для улучшения качества обслуживания клиентов. При увеличении потребностей компании возможно расширение функциональных возможностей. Решение включает в себя автоматическое распределение вызовов (ACD), мониторинг и статистика звонков, помощник агента.

Преимущества:
  • Разработан для малых и средних предприятий, которые не имеют специализированных IT специалистов
  • Комплексная, модульная система, поддержка TDM, IP телефония, доступ в Интернет, электронная почта и локальная сеть
  • Доступ в публичную сеть возможен по таким каналам как: аналоговые линии, PCM, ISDN и SIP.
  • Сеть основана на Standard Virtual Private Network (ISVPN), через IP посредством Internet, либо через SIP
  • Голосовые коммуникации, call server, голосовая почта, персональный помощник, интегрированный CTI сервер, поддержка аналоговых, цифровых и Ip абонентов, пользователей мобильных устройств и PC softphone, решения для гостиничных комплексов.
  • IP телефония позволяющая передавать голос через LAN либо глобальную сеть WAN. IP телефоны могут работать совместно с цифровыми телефонами, и предоставлять одинаковый уровень сервисов.
  • Профессиональное Internet решение, включает в себя: firewall, proxy server, cache server, IP VPN server, e-mail server, и intranet server обеспечивающий безопасность, высокоскоростной доступ через IDSN или ADSL.
  • Встроенный e-mail server, универсальное решение для обмена сообщениями и почтой реализованное при интеграции голосовой почты и e-mail, поддержка стандартных e-mail клиентов (POP3, IMAP4), доступ к голосовой почте и e-mail через простой браузер.
  • Интегрированное LAN решение, в которое включены такие сервисы как, Dynamic Host Configuration Protocol (DHCP), Domain Name System (DNS) а также файловый сервер.
  • Решения для мобильности, который предлагаются согласно ваших потребностей. Мобильные решения для настольных телефонов, bluetooth гарнитуры для комфорта и свободы передвижения. Решения для мобильности сотрудников вне зависимости от их рабочего места основанные на DECT и WIFI. Приложения для пользователей мобильных устройств Cellular Extensions.
  • Открытая технология: система основана на Linux, поддержка всех стандартов для Интернета, телефонии и передачи данных.
