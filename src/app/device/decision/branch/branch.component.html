Маршрутизатор семейства Alcatel-Lucent OmniAccess 5510 Unified Services Gateway (USG) обеспечивает безопасное высокоскоростное проводное соединение для передачи голоса и данных для маленьких офисов. Компактный, доступный по цене с фиксированной конфигурацией, OmniAccess 5510 USG обеспечивает WAN подключение, маршрутизацию, комплексную безопасность, передачу голоса по IP, видео и данные в маленький офис, филиал. Его программная платформа спроектирована на модульной основе и имеет несравнимый уровень работоспособности и надёжности.

OmniAccess 5510 USG предоставляет полный набор услуг необходимых для филиала:
  • Сеть: маршрутизация, коммутация и качество сервисов (QoS)
  • Безопасность: Firewall , определение и предотвращение вторжений (IDS/IPS) и (VPN)
  • Голос: SIP/IP Touch Application Level Gateway (ALG), локально встроенный call manager, all Admission Control (CAC),
  • Сетевые сервисы: name system (DNS), Dynamic host configuration protocol (DHCP), Radius, TACACS+
  • Управление: унифицированный интерфейс для управления всеми сервисами.
OmniAccess 5510 USG является дополнением к OmniAccess USG маршрутизатор и доступен в четырех моделях:
  • OmniAccess 5510-TE (интегрированный в один порт T1/E1 WAN-интерфейс)
  • OmniAccess 5510-SR (интегрированный в один порт WAN интерфейса (X.21, V.35, RS-232 интерфейсов, вплоть до 2 Мбит/с))
  • OmniAccess 5510-AA (интегрированный в один ADSL2 + с приложением A WAN интерфейса (ADSL через POTS))
  • OmniAccess 5510-AB (интегрированный в один порт ADSL 2 + с приложением B WAN интерфейса (ADSL через ISDN)
Каждая модель поставляется со следующими интерфейсами и аппаратными возможностями:
  • Один интегрированный интерфейс WAN - типы меняются в зависимости от модели
  • Четыре порта 10/100 switched RJ-45 Ethernet LAN
  • Один порт 10/100 routed RJ-45 Ethernet LAN, который может использоваться как Ethernet WAN
  • Поддержка электро-питания AC/DC
  • Один порт USB 2.0
  • Настольный стиль (блок на резиновых ножках), возможна опция монтажа в стойку.
  • Низкое энергопотребление «green design»
Преимущества:
  • Гибкость при выборе Т1/Е1, XDSL, Ethernet WAN интерфейсов
  • Различный форм-фактор оборудования - настольный вариант, half-rack
  • Гибкие технические и сервисные возможности
  • Глобальная обработка пакетов и объединение их в один поток
  • Отказоустойчивость, маршрутизация вызовов по PSTN при отказе WAN подключения
  • Низкая стоимость введения в эксплуатацию
